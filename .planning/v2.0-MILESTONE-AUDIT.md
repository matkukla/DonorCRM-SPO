### Partial Requirements Detail

**IMP-05: SHA256 Idempotency**
- Core requirement met: re-uploading same file does NOT reprocess. `check_duplicate_import()` finds existing batch and returns it immediately.
- Partial: `existing.status = ImportBatchStatus.DUPLICATE` set in-memory only (not saved). API response `is_duplicate` field works correctly. Import History shows "Completed" instead of "Duplicate" for repeat uploads.
- Fix: Save status to DB after setting, or leave as-is (cosmetic).

**UI-GIFT-03: Gift List Filters**
- Core requirement met: Gift list page has functional filters (search, date range, fund, owner).
- Partial: `min_amount`/`max_amount` NumberFilter fields are applied directly to `amount_cents` but frontend sends dollar values. Filter by $100 matches gifts >= $1.00.
- Fix: Multiply by 100 in backend filter or convert in frontend before sending.
