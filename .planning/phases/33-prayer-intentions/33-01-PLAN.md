    - `PrayerIntentionListCreateView(generics.ListCreateAPIView)`:
      - permission_classes = [permissions.IsAuthenticated]
      - serializer_class = PrayerIntentionSerializer
      - filterset_class = PrayerIntentionFilterSet
      - filter_backends = [DjangoFilterBackend, SearchFilter, OrderingFilter]
      - search_fields = ['title', 'contact__first_name', 'contact__last_name']

    **1. Create `frontend/src/api/prayers.ts`:**
    Define TypeScript types and API functions following the gifts.ts pattern:
    ```typescript
    export type PrayerIntentionStatus = 'active' | 'answered' | 'archived'

    export interface PrayerIntention {
      id: string
      contact: string           // contact UUID
      contact_name: string      // read-only from serializer
      title: string
      description: string
      status: PrayerIntentionStatus
      last_prayed_at: string | null
      answered_at: string | null
      archived_at: string | null
      created_at: string
      updated_at: string
    }

    export interface PrayerIntentionCreate {
      contact: string
      title: string
      description?: string
      status?: PrayerIntentionStatus
    }

    export interface PrayerIntentionUpdate {
      title?: string
      description?: string
      status?: PrayerIntentionStatus
      contact?: string
    }
    ```
    API functions (all use apiClient from @/api/client):
    - `getPrayers(params?: Record<string, string>)` — GET /api/v1/prayers/ with params
    - `getPrayer(id: string)` — GET /api/v1/prayers/{id}/
    - `createPrayer(data: PrayerIntentionCreate)` — POST /api/v1/prayers/
    - `updatePrayer(id: string, data: PrayerIntentionUpdate)` — PATCH /api/v1/prayers/{id}/
    - `deletePrayer(id: string)` — DELETE /api/v1/prayers/{id}/
    - `markPrayed(id: string)` — POST /api/v1/prayers/{id}/prayed/
    - `getTodaysFocus()` — GET /api/v1/prayers/focus/
    - `getContactPrayers(contactId: string)` — GET /api/v1/contacts/{contactId}/prayer-intentions/
    All functions return response.data as typed responses.

    - `useTodaysFocus()` — useQuery with key ['prayers', 'focus']
