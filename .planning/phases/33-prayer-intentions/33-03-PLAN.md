    **Keyboard handler:**
    ```tsx
    useEffect(() => {
      if (!open) return
      const handleKeyDown = (e: KeyboardEvent) => {
        switch (e.key) {
          case 'ArrowRight':
          case ' ':
            e.preventDefault()
            goToNext()
            break
          case 'ArrowLeft':
            e.preventDefault()
            goToPrevious()
            break
          case 'p':
          case 'Enter':
            e.preventDefault()
            handleMarkPrayed()
            break
          case 'Escape':
            e.preventDefault()
            handleClose()
            break
        }
      }
      window.addEventListener('keydown', handleKeyDown)
      return () => window.removeEventListener('keydown', handleKeyDown)
    }, [open, currentIndex])
    ```
    IMPORTANT: Call `e.preventDefault()` for ALL handled keys to prevent page scroll. Clean up listener on unmount.

    **Mark as Prayed flow:**
    - Call markPrayed mutation
    - Add intention ID to prayedIds set
    - Auto-advance to next intention
    - If last intention, go to completion screen

    - "You prayed for X intention(s) today" message
    - "Return" button that calls onClose

    **Edge cases:**
    - Empty intentions: Show "No intentions for today" with close button
    - Single intention: No previous/next, just mark and complete
