DECISION (Claude's discretion, per RESEARCH.md "Passing full request is more flexible"): Update the _scope_* helper signatures to accept request=None and pass it through. Update service functions that call them to also accept and forward request. Then update the view call sites to pass request.

SPECIAL HANDLING — dashboard/services.py:

Only the two direct view-level get_visible_user_ids() calls in dashboard/views.py (lines 49 and 257) need the request passed.

    def get_visible_user_ids(user, request=None):
        """Return set of user IDs whose data this user can see, or None for 'all'.
        ...

        # View As override: scope to the target user regardless of viewer role.
        if request is not None:
            view_as_user = getattr(request, 'view_as_user', None)
            if view_as_user is not None:
                return {view_as_user.id}

        if user.role in ['finance', 'read_only']:
            return None
        if user.role == 'coach':
            ids = set(
                user.coached_users
                .filter(is_active=True)
                .values_list('id', flat=True)
            )
            ids.add(user.id)
            return ids
        return {user.id}
