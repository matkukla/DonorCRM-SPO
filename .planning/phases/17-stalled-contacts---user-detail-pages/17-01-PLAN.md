**Pagination state:**
- Add `pageIndex` state (starting at 0) and a `PAGE_SIZE` constant of 50
- Compute `offset` as `pageIndex * PAGE_SIZE`
- Compute `pageCount` as `Math.ceil((data?.total_count || 0) / PAGE_SIZE)`
- Pass `{ limit: PAGE_SIZE, offset, sort_by: sortBy, sort_dir: sortDir }` to `useAdminStalledContacts`

**Sorting state:**
- Add `sortBy` state (default: `'days_stalled'`) and `sortDir` state (default: `'desc'`)
- Create `handleSortChange(column: string)` function:
  - If same column: toggle sortDir between 'asc' and 'desc'
  - If different column: set sortBy to new column, reset sortDir to 'desc'
  - Always reset pageIndex to 0 (pitfall from research: pagination state reset on sort change)

- Make "Contact Name" (sort_by: `full_name`), "Owner" (sort_by: `owner_name`), and "Days Stalled" (sort_by: `days_stalled`) columns clickable
- Use a button with `className="flex items-center gap-1 hover:text-foreground cursor-pointer"` inside TableHead
- Import `ArrowUpDown` from `lucide-react` for sort indicator icon (size h-4 w-4)
- "Last Activity" and "Status" columns remain non-sortable (backend does not support sorting by status)

**Pagination controls:**
- Add a div below the table with flex justify-between items-center
- Left side: "Showing {offset + 1}-{Math.min(offset + PAGE_SIZE, total_count)} of {total_count} contacts"

**Important details:**
