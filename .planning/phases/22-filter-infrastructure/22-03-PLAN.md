**Refactor ContactList.tsx to use the new filter infrastructure instead of manual useSearchParams.**

**Step 1: Replace manual URL state with useFilterParams**

Current pattern (REMOVE):
```tsx
const [searchParams, setSearchParams] = useSearchParams()
const page = parseInt(searchParams.get("page") || "1", 10)
const search = searchParams.get("search") || ""
const status = searchParams.get("status") as ContactStatus | undefined
const needsThankYou = searchParams.get("needs_thank_you") === "true"
```

New pattern (ADD):
```tsx
import { useFilterParams, contactFilterParsers } from '@/hooks/useFilterParams'
import { contactPresets } from '@/lib/filter-presets'
import { FilterBar } from '@/components/shared/FilterBar'
import { ExportCSVButton } from '@/components/shared/ExportCSVButton'

const {
  filters,
  setFilters,
  clearAll,
  activeFilters,
  activeFilterCount,
  toQueryParams,
} = useFilterParams(contactFilterParsers)
```

Access filter values directly: `filters.page`, `filters.status`, `filters.needs_thank_you`, etc.

**Step 2: Update the useContacts hook call**

Pass the filter values to the hook. The existing `useContacts` hook accepts params like `{ page, page_size, search, status, needs_thank_you }`. Update the call to use `filters` object:

```tsx
const { data, isLoading } = useContacts({
  page: filters.page,
  page_size: PAGE_SIZE,
  search: filters.search || undefined,
  status: filters.status || undefined,
  needs_thank_you: filters.needs_thank_you ?? undefined,
  last_gift_after: filters.last_gift_after || undefined,
  last_gift_before: filters.last_gift_before || undefined,
  group: filters.group || undefined,
})
```

**Step 3: Update useContacts hook to accept new filter params**

In `frontend/src/hooks/useContacts.ts`, add `last_gift_after`, `last_gift_before`, and `group` to the params type and include them in the query params passed to the API. Also include them in the React Query key so the cache updates when filters change.

**Step 4: Replace manual filter handlers with FilterBar**

Remove the individual `handleStatusFilter`, `handleThankYouFilter` functions and the manual dropdown menus for filtering. Replace with a FilterBar component:

```tsx
<FilterBar
  activeFilters={activeFilters}
  onClearAll={clearAll}
  onRemoveFilter={(key) => setFilters({ [key]: null, page: 1 })}
  filterLabels={{
    status: "Status",
    needs_thank_you: "Needs Thank You",
    last_gift_after: "Gift From",
    last_gift_before: "Gift To",
    group: "Group",
  }}
  filterValueLabels={{
    status: {
      prospect: "Prospect",
      donor: "Donor",
      lapsed: "Lapsed",
      major_donor: "Major Donor",
      deceased: "Deceased",
    },
  }}
  presets={contactPresets}
  onApplyPreset={(preset) => setFilters({ ...preset.getParams(), page: 1 })}
  exportUrl="/api/v1/contacts/export/csv/"
  exportParams={toQueryParams()}
>
  {/* Filter controls as children */}
  <div className="flex flex-wrap items-center gap-2">
    {/* Search input */}
    <form onSubmit={handleSearch} className="flex gap-2">
      <Input ... />
    </form>

    {/* Status dropdown */}
    <DropdownMenu>
      <DropdownMenuTrigger asChild>
        <Button variant="outline" size="sm">
          <Filter className="h-4 w-4 mr-1" />
          Status
        </Button>
      </DropdownMenuTrigger>
      <DropdownMenuContent>
        <DropdownMenuItem onClick={() => setFilters({ status: null, page: 1 })}>All</DropdownMenuItem>
        <DropdownMenuItem onClick={() => setFilters({ status: 'prospect', page: 1 })}>Prospect</DropdownMenuItem>
        <DropdownMenuItem onClick={() => setFilters({ status: 'donor', page: 1 })}>Donor</DropdownMenuItem>
        ...
      </DropdownMenuContent>
    </DropdownMenu>

    {/* Thank You toggle button */}
    <Button
      variant={filters.needs_thank_you ? "default" : "outline"}
      size="sm"
      onClick={() => setFilters({
        needs_thank_you: filters.needs_thank_you ? null : true,
        page: 1,
      })}
    >
      <Heart className="h-4 w-4 mr-1" />
      Needs Thank You
    </Button>
  </div>
</FilterBar>
```
