Purpose: The Import Center UI needs to display last import date/status for each tile and show dependency warnings when attempting Transaction/Pledge imports without Funds/Entities.

Output: GET /api/v1/imports/runs/latest/ endpoint returning latest import runs and dependency counts.

<task type="auto">
  <name>Task 1: Create LatestImportRunsView API endpoint</name>
  <files>apps/imports/views.py</files>
  <action>
Add LatestImportRunsView class to apps/imports/views.py:

1. Import IsAdmin from apps.core.permissions (already imported for other views)
2. Create class LatestImportRunsView(APIView) with:
   - permission_classes = [permissions.IsAuthenticated, IsAdmin]
   - GET method that:
     a. Queries ImportRun for latest run per type using ImportType.values iteration
     b. For each type, get most recent run ordered by -created_at, .first()
     c. Returns dict with type as key, run data (id, status, created_at, created_count, updated_count, error_count) or null as value
     d. Also includes dependency counts:
        - funds_count: Fund.objects.count()
        - entities_with_external_id_count: Contact.objects.exclude(external_id='').exclude(external_id__isnull=True).count()

Response format:
```json
{
  "funds": {"id": "...", "status": "completed", "created_at": "...", "created_count": 10, "updated_count": 5, "error_count": 0} | null,
  "entities": {...} | null,
  "transactions": {...} | null,
  "pledges": {...} | null,
  "dependency_counts": {
    "funds_count": 15,
    "entities_with_external_id_count": 100
  }
}
```

Use str(run.id) for UUID serialization.
  </action>
  <verify>python manage.py shell -c "from apps.imports.views import LatestImportRunsView; print('View exists')"</verify>
  <done>LatestImportRunsView class exists with GET method returning latest runs and dependency counts</done>
</task>
