```python
class BroadcastTask(TimeStampedModel):
    """Parent record for a broadcast task distribution."""
    sender = models.ForeignKey(
        settings.AUTH_USER_MODEL,
        on_delete=models.CASCADE,
        related_name='sent_broadcasts',
        db_index=True,
    )
    title = models.CharField(max_length=255)
    description = models.TextField(blank=True)
    task_type = models.CharField(
        max_length=20,
        choices=TaskType.choices,
        default=TaskType.OTHER,
    )
    priority = models.CharField(
        max_length=20,
        choices=TaskPriority.choices,
        default=TaskPriority.MEDIUM,
    )
    due_date = models.DateField()
    target_type = models.CharField(
        max_length=30,
        choices=[
            ('all_missionaries', 'All Missionaries'),
            ('all_supervisors', 'All Supervisors'),
            ('specific_users', 'Specific Users'),
            ('my_team', 'My Team'),
        ],
    )
    recipient_ids = models.JSONField(default=list)
    recipient_count = models.PositiveIntegerField(default=0)
    is_cancelled = models.BooleanField(default=False)
    cancelled_at = models.DateTimeField(null=True, blank=True)

    class Meta:
        db_table = 'broadcast_tasks'
        ordering = ['-created_at']
        indexes = [
            models.Index(fields=['sender', '-created_at']),
        ]

    def __str__(self):
        return f'Broadcast: {self.title} ({self.recipient_count} recipients)'
```

Create apps/tasks/broadcast_services.py with four functions:

1. `resolve_recipients(sender, target_type, specific_user_ids=None)` -> list[User]:
   - `all_missionaries`: assert sender.role == 'admin', return User.objects.filter(role='missionary', is_active=True)
   - `all_supervisors`: assert sender.role == 'admin', return User.objects.filter(role='supervisor', is_active=True)
   - `my_team`: assert sender.role in ('admin', 'supervisor'), return sender.supervised_users.filter(is_active=True)
   - `specific_users`: fetch User.objects.filter(id__in=specific_user_ids, is_active=True). If sender.role == 'supervisor', intersect with sender.supervised_users.filter(is_active=True).values_list('id', flat=True). Return list(users).
   - Default: return []

2. `create_broadcast(sender, title, description, task_type, priority, due_date, target_type, specific_user_ids=None)` -> BroadcastTask:
   - Wrapped in @transaction.atomic
   - Call resolve_recipients(...) to get recipient list
   - If no recipients: raise ValidationError("No valid recipients found")
   - Create BroadcastTask with recipient_ids=[str(u.id) for u in recipients], recipient_count=len(recipients)
   - bulk_create Task objects: one per recipient with owner=recipient, broadcast=broadcast, title/description/task_type/priority/due_date from broadcast, status=TaskStatus.PENDING, contact=None, journal=None
   - Return broadcast

3. `update_broadcast(broadcast, title=None, description=None, task_type=None, priority=None, due_date=None)` -> BroadcastTask:
   - Wrapped in @transaction.atomic
   - Build update_fields dict from non-None params
   - Update BroadcastTask fields and save
   - Build Task update dict from same non-None params
   - Task.objects.filter(broadcast=broadcast).exclude(status=TaskStatus.COMPLETED).exclude(status=TaskStatus.CANCELLED).update(**task_update_fields) — cascade to incomplete copies only
   - Return broadcast

4. `cancel_broadcast(broadcast)` -> BroadcastTask:
   - Wrapped in @transaction.atomic
   - Delete incomplete copies: Task.objects.filter(broadcast=broadcast).exclude(status=TaskStatus.COMPLETED).delete()
   - Set broadcast.is_cancelled = True, broadcast.cancelled_at = timezone.now(), save
   - Return broadcast
