    parse_entities_csv(file_content: str, user) -> Tuple[List[dict], List[dict]]:
    - Input: CSV with columns entity_id, name, email, phone, address, entity_type
    - Returns (valid_records, errors) tuple

    - Empty entity_id -> error "entity_id is required"

    - Duplicate entity_id in file -> error "Duplicate entity_id in file: X"
    - entity_id > 100 chars -> error "exceeds maximum length"
    - name > 300 chars -> error "exceeds maximum length"
    - Invalid email format -> error "Invalid email format"

    - Optional fields (email, phone, address) can be empty
    - address maps to street_address only

    - Creates new Contacts with owner=user, external_id from entity_id
    - Updates existing Contacts with matching owner+external_id
    - Uses bulk_create(unique_fields=['owner', 'external_id']) - CRITICAL: use 'owner' not 'owner_id'
    - Returns (created_count, updated_count)
    - Updates import_run with counts and status=COMPLETED

**parse_entities_csv function:**
- Signature: `def parse_entities_csv(file_content: str, user) -> Tuple[List[dict], List[dict]]:`
- Use csv.DictReader(io.StringIO(file_content))
- Validate required columns: entity_id, name (return early if missing)
- Track seen_entity_ids set for duplicate detection
- For each row:
  - Validate entity_id (required, max 100 chars, no formula prefix, no duplicate)
  - Validate name (required, max 300 chars, no formula prefix)
  - Split name: `parts = name.split(); first_name = ' '.join(parts[:-1]) if len(parts) > 1 else parts[0]; last_name = parts[-1] if len(parts) > 1 else ''`
  - Validate email (optional, use _validate_email, max 254 chars)
  - Validate phone (optional, max 20 chars)
  - Validate address (optional, max 255 chars, maps to street_address)
  - IGNORE entity_type column (no Contact field for it)
- Return (valid_records, errors) with valid_records containing:
  - entity_id, first_name, last_name, email, phone, street_address

**import_entities function:**
- Signature: `def import_entities(records: List[dict], user, import_run) -> Tuple[int, int]:`
- Handle empty records: set import_run status=COMPLETED, counts=0, return (0, 0)
- Query existing external_ids for this user:
  `existing = set(Contact.objects.filter(owner=user, external_id__in=[r['entity_id'] for r in records]).values_list('external_id', flat=True))`
- Build Contact objects list with owner=user for each record
- Use `Contact.objects.bulk_create(contact_objects, update_conflicts=True, unique_fields=['owner', 'external_id'], update_fields=['first_name', 'last_name', 'email', 'phone', 'street_address'])`
- CRITICAL: unique_fields must be ['owner', 'external_id'] NOT ['owner_id', 'external_id']
- Calculate counts: created = len([r for r in records if r['entity_id'] not in existing])
- Update import_run: created_count, updated_count, status=ImportStatus.COMPLETED
- Return (created_count, updated_count)

DO NOT include 'owner' or 'external_id' in update_fields.
