### E2E User Flows (9/9)

1. **Dashboard Overview** — Admin navigates to /admin/analytics → sees summary cards, trend charts, funnel, activity table, alerts, heatmap, comparisons
2. **Stalled Contacts Monitoring** — Admin views stalled contacts → paginate → sort → filter by date → export CSV
3. **User Detail Inspection** — Admin clicks user → sees metrics, trends, journal list
4. **Funnel Drill-Down** — Admin clicks funnel segment → slide-in panel shows contacts in that stage
5. **User Quick View** — Admin clicks Quick View in activity table → slide-in panel shows user stats
6. **Date Range Filtering** — Admin selects date preset → all widgets update → URL syncs
7. **CSV Export** — Admin exports stalled contacts or team activity → file downloads with date range in filename
8. **Time Period Comparison** — Admin selects date range → sees current vs prior period with trend arrows
9. **User Comparison** — Admin selects two missionaries → sees side-by-side metrics
