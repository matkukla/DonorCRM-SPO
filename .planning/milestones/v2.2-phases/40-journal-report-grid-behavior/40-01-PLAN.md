Add a new `@action` method to `JournalAnalyticsViewSet`:

```python
@action(detail=False, methods=['get'], url_path='journal-report')
def journal_report(self, request):
```

Implementation requirements:
1. **Parameters:** `journal_id` (required), `date_from` (optional), `date_to` (optional) from `request.query_params`
2. **Validation:** Return 400 if `journal_id` missing. Look up `Journal` with `get_object_or_404`. Return 403 if user is not admin and not the journal owner.
3. **Base querysets** scoped to the journal:
   - `members = JournalContact.objects.filter(journal=journal)`
   - `decisions = Decision.objects.filter(journal_contact__journal=journal)`
   - `events = JournalStageEvent.objects.filter(journal_contact__journal=journal)`
   - `next_steps = NextStep.objects.filter(journal_contact__journal=journal)`
4. **Date filtering:** If `date_from` provided, filter `events` and `decisions` by `created_at__date__gte=date_from`. If `date_to` provided, filter by `created_at__date__lte=date_to`. Do NOT date-filter `members` (Total Contacts is a current snapshot) or `next_steps` (stalled/open is current state).
5. **Metric cards:**
   - `total_contacts`: `members.count()`
   - `with_decisions`: `decisions.exclude(status='declined').values('journal_contact').distinct().count()`
   - `confirmed_amount`: `decisions.filter(status='active').aggregate(total=Sum('amount'))['total'] or 0` -- convert to `str()`
   - `pending_amount`: `decisions.filter(status='pending').aggregate(total=Sum('amount'))['total'] or 0` -- convert to `str()`
6. **Contacts by stage** (bar chart data): Annotate `members` with a `Subquery` to get each member's latest stage from `JournalStageEvent` (ordered by `-created_at`, take first `.values('stage')[:1]`). Then group by `current_stage` and count. Use `OuterRef('pk')`. Return as list of `{'stage': ..., 'count': ...}`. Contacts with no events should have `current_stage=None` -- map these to `'none'` in the output.
7. **Decision status distribution** (donut chart data): `decisions.values('status').annotate(count=Count('id'))` -- return as list of `{'status': ..., 'count': ...}`.
8. **Alerts:**
   - `stalled_contacts`: Members with no `stage_events` with `created_at__gte` 30 days ago. Use `members.exclude(stage_events__created_at__gte=stall_threshold).count()` where `stall_threshold = timezone.now() - timedelta(days=30)`.
   - `open_next_steps`: `next_steps.filter(completed=False).count()`
9. **Response shape:**
```python
return Response({
    'metrics': {
        'total_contacts': total_contacts,
        'with_decisions': with_decisions,
        'confirmed_amount': str(confirmed_amount),
        'pending_amount': str(pending_amount),
    },
    'goal_amount': str(journal.goal_amount),
    'stage_distribution': list(stage_distribution),
    'decision_status': list(decision_status),
    'alerts': {
        'stalled_contacts': stalled_count,
        'open_next_steps': open_next_steps,
    }
})
```

The URL route is automatically handled by the existing router registration (`router.register(r'analytics', JournalAnalyticsViewSet, ...)`). The `url_path='journal-report'` makes the endpoint available at `/api/journals/analytics/journal-report/`. No changes needed to `urls.py`.

```typescript
/** Get journal report data (scoped to single journal, with optional date range) */
export async function getJournalReport(
  journalId: string,
  dateParams: { date_from?: string; date_to?: string } = {}
): Promise<JournalReportData> {
  const params = new URLSearchParams()
  params.append('journal_id', journalId)
  if (dateParams.date_from) params.append('date_from', dateParams.date_from)
  if (dateParams.date_to) params.append('date_to', dateParams.date_to)
  const response = await apiClient.get<JournalReportData>(
    `/journals/analytics/journal-report/?${params.toString()}`
  )
  return response.data
}
```
