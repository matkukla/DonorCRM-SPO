# Plan 43-04: Admin Assignments Page

## Objective

Build `AdminAssignments.tsx` — a new admin page with an inline-editable table where admins can assign supervisors and coaches to each missionary, with dirty tracking, bulk operations, and a diff-based save.

## Tasks

### Task 1: Create frontend/src/pages/admin/AdminAssignments.tsx

**Dependencies:** Uses `useAssignments` and `useUpdateAssignments` hooks from Task 8 of 43-03. Uses shadcn `Select`, `Input`, `Button`, `Checkbox`, `Badge`, `toast` components.

**State:**

**Layout:**
```
<PageHeader title="Assignments" subtitle="Assign supervisors and coaches to each missionary" />

Toolbar row:
  <Input placeholder="Search missionaries..." value={search} onChange={...} />
  <Button disabled={dirty.size === 0} onClick={handleSave}>
    Save Changes {dirty.size > 0 && `(${dirty.size})`}
  </Button>

Bulk bar (visible when selectedIds.size > 0):
  "{selectedIds.size} selected"
  Supervisor: <Select ...> [Apply]
  Coach: <Select ...> [Apply]
  <Button variant="ghost">Clear selection</Button>

Table:
  columns: [checkbox, Missionary, Email, Supervisor (dropdown), Coach (dropdown), Status]

  Each row:
    - Checkbox to toggle selectedIds
    - Missionary name + email
    - Supervisor <Select>: "— Unassigned" + supervisors list
    - Coach <Select>: "— Unassigned" + coaches list
    - Status: "●" dot (visible when row is in dirty set)

  When supervisor or coach dropdown changes:
    - Update localAssignments map
    - Add missionary id to dirty set
```

**Save handler:**
```typescript
const handleSave = async () => {
  const assignments = [...dirty].map(id => ({
    missionary_id: id,
    ...localAssignments.get(id)!,
  }))
  await updateAssignmentsMutation.mutateAsync(assignments)
  setDirty(new Set())
  toast({ title: "Assignments saved" })
}
```

Handle per-row errors: if `errors.length > 0`, show error toast listing them.

**Bulk apply handler:**
```typescript
const handleBulkApply = (field: 'supervisor_id' | 'coach_id', value: string | null) => {
  setLocalAssignments(prev => {
    const next = new Map(prev)
    selectedIds.forEach(id => {
      const current = next.get(id) || { supervisor_id: null, coach_id: null }
      next.set(id, { ...current, [field]: value })
    })
    return next
  })
  setDirty(prev => new Set([...prev, ...selectedIds]))
}
```

**Search filter:**
```typescript
const filteredMissionaries = missionaries.filter(m =>
  m.full_name.toLowerCase().includes(search.toLowerCase()) ||
  m.email.toLowerCase().includes(search.toLowerCase())
)
```

**Select all checkbox:** checks all filtered missionaries.
