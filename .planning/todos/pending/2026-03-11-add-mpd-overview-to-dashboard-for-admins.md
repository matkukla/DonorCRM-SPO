## Problem

Admin users need visibility into MPD (Ministry Partner Development) data at a glance. The dashboard currently lacks an MPD Overview section, meaning admins must navigate to individual reports to get a high-level picture of partner development status across missionaries.

## Solution

Add an MPD Overview tile/section to the Dashboard that is visible only to Admin roles. Likely sourced from the Smartsheet MPD data import. Content could include metrics like total active partners, funding percentages, or recent MPD activity.
