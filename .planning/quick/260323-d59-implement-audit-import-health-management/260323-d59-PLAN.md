  <action>
    Create `apps/imports/management/commands/audit_import_health.py` as a BaseCommand subclass.

    **Arguments:** `--verbose` (store_true, show per-record detail) and `--json` (store_true, output JSON).

    **handle() method flow:**

    1. Edge case check: if Solicitor.objects.count() == 0, print "No solicitors found. Import solicitors first." and return.

    2. Collect all report data into a dict (for both text and JSON output).

    3. **Section 1 - Solicitor Linking:**
       - total = Solicitor.objects.count()
       - linked = Solicitor.objects.filter(user__isnull=False).count()
       - unlinked = total - linked
       - linked_pct = round(linked / total * 100, 1) if total else 0
       - unlinked_pct = round(unlinked / total * 100, 1) if total else 0
       - Near-miss detection: for each unlinked solicitor, compare normalized_name against:
         a) User.objects.values_list('first_name', 'last_name') — build "First Last" strings
         b) MissionaryAlias.objects.values_list('source_name', flat=True)
         Use `difflib.SequenceMatcher(None, a.lower(), b.lower()).ratio() >= 0.8` for fuzzy matching.
         Collect near-misses as list of {solicitor_name, candidate, score}.

    4. **Section 2 - Contact Ownership:**
       - Group contacts by owner__role using .values('owner__role').annotate(count=Count('id')).
       - Misattribution detection: find contacts where owner.role in ('admin', 'supervisor') AND the contact has gifts with credits where credit.solicitor.user.role == 'missionary'. Query:
         ```python
         Contact.objects.filter(
             owner__role__in=['admin', 'supervisor'],
             gifts__credits__solicitor__user__role='missionary'
         ).distinct().select_related('owner')
         ```
         For each misattributed contact, record {contact_name, owner_name, owner_role, missionary_solicitor_name}.

    5. **Section 3 - Gift Credit Integrity:**
       - total_gifts = Gift.objects.count()
       - total_credits = GiftCredit.objects.count()
       - orphaned_gifts = Gift.objects.annotate(credit_count=Count('credits')).filter(credit_count=0)
       - orphaned_count = orphaned_gifts.count()
       - unlinked_value = orphaned_gifts.aggregate(total=Sum('amount_cents'))['total'] or 0
       - unlinked_dollars = Decimal(unlinked_value) / Decimal(100)

    6. **Section 4 - Recurring Gift Credit Integrity:**
       - Same pattern as Section 3 but filtered to RecurringGift.objects.filter(status='active').
       - total_active = filtered.count()
       - total_credits = RecurringGiftCredit.objects.filter(recurring_gift__status='active').count()
       - orphaned = filtered.annotate(credit_count=Count('credits')).filter(credit_count=0)
       - orphaned_count, unlinked_dollars as above.

    7. **Section 5 - Dashboard Impact Estimate:**
       - missionaries = User.objects.filter(role='missionary')
       - For each missionary:
         - contacts_count = Contact.objects.filter(owner=user).count()
         - gift_sum = Gift.objects.filter(donor_contact__owner=user).aggregate(total=Sum('amount_cents'))['total'] or 0
         - recurring_sum = RecurringGift.objects.filter(donor_contact__owner=user, status='active').aggregate(total=Sum('amount_cents'))['total'] or 0
         - credit_count = GiftCredit.objects.filter(solicitor__user=user).count()
         - Flag if (contacts_count == 0 and credit_count > 0) or (gift_sum == 0 and credit_count > 0)
       - Collect per-missionary rows and flagged issues.

    8. **Verdict:**
       - Count issues: unlinked solicitors + misattribution count + orphaned gift count + orphaned recurring count + flagged missionary count.
       - If 0 issues: "HEALTHY — All import data is clean."
       - Else: "NEEDS ATTENTION — {N} issue(s) detected."

    **Output formatting:**
    - Default (text): Use self.stdout.write with self.style.SUCCESS/WARNING/ERROR. Format each section with a header line (e.g., "=== Section 1: Solicitor Linking ==="), then key-value stats.
    - --verbose: After each section's summary stats, print per-record detail lists (e.g., each unlinked solicitor name, each misattributed contact).
    - --json: Build a dict with keys 'solicitor_linking', 'contact_ownership', 'gift_credit_integrity', 'recurring_gift_credit_integrity', 'dashboard_impact', 'verdict'. Print via json.dumps(data, indent=2, default=str). Skip all styled text output.

    **Style reference:** Follow reconcile_missionaries.py pattern — BaseCommand, help string, add_arguments, self.stdout.write with self.style.
